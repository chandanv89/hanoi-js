Embed Font

To embed your selected fonts into a webpage, copy this code into the <head> of your HTML document.
1. Import the stylesheet for the webfonts:
	Sandard Method:
		Specify in your HTML:
			<link href="https://fonts.googleapis.com/css?family=Fira+Mono|Fira+Sans:400,400i|Playfair+Display:400,400i,900,900i|Slabo+27px" rel="stylesheet">
				
	@IMPORT Method:
		Specify import clause either in yhour markup or in the stylesheet:
			@import url('https://fonts.googleapis.com/css?family=Fira+Mono|Fira+Sans:400,400i|Playfair+Display:400,400i,900,900i|Slabo+27px');
	
2. Specify in CSS - Use the following CSS rules to specify these families:
	[Common to both the import methods mentioned above]
	font-family: 'Fira Sans', sans-serif;
	font-family: 'Fira Mono', monospace;
	font-family: 'Slabo 27px', serif;
	font-family: 'Playfair Display', serif;

	For examples of how fonts can be added to webpages, see the getting started guide.
